# Choco Chat 利用法

**（前文）**  
本法は、Choco Chat における秩序の維持および利用者間の円滑な交流を確保することを目的とし、利用者は本法の定めに従う義務を負う。

---

## **第1章　総則**

### **第1条（目的）**  
本法は、Choco Chat の健全な運営を図るため、禁止行為、利用者の権利義務、及び違反時の措置について定める。

### **第2条（定義）**  
本法において使用する用語の定義は、次の各号に掲げるとおりとする。  
1. 「利用者」とは、Choco Chat にアクセスし、閲覧または発言を行う者をいう。  
2. 「運営」とは、Choco Chat の管理権限を有する者をいう。  
3. 「サービス」とは、Choco Chat において提供される全ての機能をいう。

---

## **第2章　禁止行為**

### **第3条（他者への不当な行為の禁止）**  
利用者は、次に掲げる行為を行ってはならない。  
1. 侮辱、名誉毀損、脅迫その他、他者に精神的損害を与える行為  
2. 過度な性的表現、暴力的表現、または一般的に不快と認められる発言  
3. 差別的言動、誹謗中傷、挑発行為  
4. その他、運営が不適切と判断する行為

### **第4条（迷惑行為の禁止）**  
利用者は、次に掲げる迷惑行為を行ってはならない。  
1. ログイン・ログアウトを短時間に反復する行為  
2. スパム投稿、荒らし行為  
3. 会話の進行を妨害する目的で行われる一切の行為

---

## **第3章　コミュニケーションに関する規定**

### **第5条（話し方の自由）**  
1. 利用者は、敬語・口語・タメ口その他任意の話し方を選択することができる。  
2. ただし、前章に定める禁止行為に該当する内容は許されない。

### **第6条（役職と立場の平等）**  
1. Choco Chat における役職の有無は、利用者間の上下関係を形成しない。  
2. すべての利用者は、互いを対等な存在として尊重しなければならない。

---

## **第4章　罰則**

### **第7条（措置の種類）**  
利用者が本法に違反した場合、運営は次の措置を講ずることができる。  
1. **警告（Warning）**：軽微な違反に対する注意  
2. **発言停止（Mute）**：一定期間、発言権を停止する措置  
3. **利用停止（Ban）**：アカウントの利用を禁止する措置  
4. **IPアドレスによる接続遮断（IP BAN）**：  
　悪質性が高い、または重大な損害を与えたと認められる場合に適用し、  
　同一ネットワークからのアクセスを全面的に禁止する

### **第8条（措置の基準）**  
1. 運営は、違反行為の内容、悪質性、継続性、過去の違反歴を考慮して措置を決定する。  
2. 措置は、利用者の故意・過失を問わず適用される場合がある。

---

## **第5章　附則**

### **第9条（規約の改定）**  
1. 運営は、必要と認める場合、本法を改定することができる。  
2. 利用者は、改定後もサービスを利用することで、改定内容に同意したものとみなす。
